Long Server is the over-arching program for Long Now's digital continuity software projects. We began with several research projects into pervasive server and email infrastructure in 01999 sponsored by Mitchell Kapor and David Liddle. Most recently we released an open source Timeline tool dubbed the Long View, and are now working on a file format conversion project called The Format Exchange.
